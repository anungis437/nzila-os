# MVP Feature List

**Owner:** Aubert

### **1. MVP Feature Tiers**

Memora’s MVP launches a **balanced, rest-aware experience** for patients, caregivers, and clinics — delivering **emotional value**, **clinical trust**, and **sustained engagement** without overloading users. All systems are designed for **scalability**, **compliance**, and **adaptive pacing** from day one.

| Tier | Description |
| --- | --- |
| **Core (Must-Have)** | Essential to Memora’s promise of trust, reflection, and healthy interaction cycles |
| **Supportive (Should-Have)** | Enhances comfort, accessibility, and value for secondary users |
| **Deferred (Post-MVP)** | Features for broader scale and richer personalization beyond initial rollout |

---

### ✅ **2. CORE MVP FEATURES**

| Feature | Description | Stakeholder Impact | Status |
| --- | --- | --- | --- |
| **Adaptive Cognitive Game Suite v1** | 3 games (Memory Match, Sequence Builder, Category Sorter) paced to avoid repetition and support light-reflection play | Patients | ✅ Built & Tested |
| **Companion AI v1** | Empathetic, non-intrusive Companion that modulates tone, initiates rest breaks, and avoids guilt-based prompts | Patients, Caregivers | ✅ Scripted + Embedded |
| **Caregiver Access Portal** | Linked view that prioritizes connection, not oversight — showing meaningful use, not micromanagement | Caregivers | ✅ Integrated |
| **Clinic Dashboard v1** | Shows adherence trends, quiet cycles, and “soft engagement” metrics (not raw completions) | Clinics | ⏳ Final testing |
| **Routine & Rest System** | Encourages daily interaction with built-in pauses, variable prompts, and "light day" logic | Patients | ✅ Embedded in Companion |
| **Bilingual UX** | English/French UI with Companion behavior localized to cultural tone sensitivity | All | ✅ Ready |
| **Companion-Guided Onboarding** | Calm, welcoming walkthrough designed to reduce cognitive load and build early trust | All | ✅ Finalized |
