1. Purpose

This toolkit provides a standardized set of onboarding materials, training content, and support documentation for memory clinics participating in a Memora pilot. It is designed to:

  • Reduce clinic staff onboarding time
  • Clarify patient consent and engagement workflows
  • Ensure regulatory alignment (e.g., Law 25, PIPEDA)
  • Enable efficient pilot deployment with minimal support requests

3. Key Workflow Summary (For Clinics)

1. Patient is informed about Memora by clinic staff
2. Clinic staff uses Onboarding Script + Flyer to describe product
3. Patient installs app or receives assisted onboarding
4. Patient completes consent steps (App or Tablet)
5. Clinic views patient engagement via dashboard
6. Staff receive optional usage flags (e.g., no session in 3+ days)
7. Data reviewed, pilot KPI captured, report template populated
