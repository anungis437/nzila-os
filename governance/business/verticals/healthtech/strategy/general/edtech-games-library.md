# EdTech Games Library

**Owner:** Aubert

*Designed for learners and families using MyLearning Companion*

### üß† **Design Framework**

- Based on **Self-Determination Theory**: Encourages intrinsic motivation via choice, mastery, and connection
- Informed by **Cognitive Load Theory**: Prevents overstimulation and supports working memory
- Embeds **Social-Emotional Learning (SEL)**: Fosters reflection, emotional literacy, and empathy
- Built for **daily use**: Short play loops, mindful rest cycles, habit-forming sequences
