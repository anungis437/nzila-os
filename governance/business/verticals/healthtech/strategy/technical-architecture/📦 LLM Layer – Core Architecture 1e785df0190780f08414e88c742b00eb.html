üß™ Prompt Lifecycle

  1. Intent Classification
    • Based on product (e.g., Memora vs. MyLearning) and user role
    • Flags urgent, personal, educational, or emotional tones
  1. System Prompt Injection
    • Adds product-specific context, tone shaping, trust anchors
    • Includes safety overrides and memory scaffolding
  1. User Message Embedding
    • Sanitized and augmented with metadata (age, mood, pacing rules)
    • Language-specific filters and token budgeting applied
  1. Model Call Execution
    • Via selected endpoint (OpenAI/Azure/local)
    • Caching layer for repeated or static responses
  1. Output Processing
    • Checks for hallucination triggers, flags sensitive content
    • Adds timestamp, sentiment score, trace ID for audit trail
