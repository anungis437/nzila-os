# Nzila Ventures - Board Report Template

This is a template for quarterly board reports. Use the export manager to generate reports.

## Usage

```bash
python -m analytics.generators.export_manager --template board_report --format markdown
```

## Report Sections

1. Executive Summary
2. Portfolio Performance
3. Financial Summary
4. Technical Progress
5. Risk Assessment
6. Strategic Priorities
7. Upcoming Milestones
